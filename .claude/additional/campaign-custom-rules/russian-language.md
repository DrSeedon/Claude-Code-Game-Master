## rules

### Language Policy
- ALL narrative text, descriptions, dialogue, NPC names, location names, quest names, item names — in Russian.
- Node IDs in world.json stay English/transliterated (type:kebab-name format).
- Tool commands (bash tools/dm-world.sh etc.) — English.
- Rules files (.md in .claude/) — English (they are code).
- Module code, lib/, infrastructure scripts — English.
- Node `name` fields: Russian.
- Descriptive text in node `data`: Russian.
- Mechanical values (dice, numbers, IDs): as-is.
